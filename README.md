# SOC Writeups 🔐

This repository contains SOC-style incident investigations and security analysis based on real-world simulated cyber attacks.

## 📌 Contents
- Phishing email analysis
- Malware investigation
- Brute force attack detection
- Unauthorized access investigations
- Web attack analysis

## 🧪 Tools & Platforms
- LetsDefend SOC platform
- Log analysis techniques
- Threat intelligence sources (VirusTotal, etc.)

## 📊 What I demonstrate
- Incident detection and response
- Log analysis and interpretation
- Identification of Indicators of Compromise (IOCs)
- Security recommendations and remediation steps

## 📂 Structure
Each case study includes:
- Incident overview
- Log analysis
- Evidence (screenshots)
- Findings
- Conclusion
- Recommendations

## 🎯 Goal
To build hands-on SOC analyst skills and document real incident response workflows for cybersecurity roles.
